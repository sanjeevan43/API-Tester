## 🚀 How to Use

### Installation
1. Open VS Code
2. Search "API Sentinel" in Extensions
3. Click Install

### Configuration
1. Click API Sentinel icon
2. Enter Base URL
3. Add API key (optional)
4. Click "SAVE SETTINGS"

### Testing
1. Click "Discover Endpoints"
2. Click "Execute Automation"
3. View results in dashboard
